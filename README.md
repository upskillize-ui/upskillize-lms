# 🎓 Upskillize LMS Portal - Complete Setup Guide

Full-Stack Learning Management System with React, Node.js, Express, and MySQL.

## Quick Start

### Prerequisites
- Node.js 18+
- MySQL 8+

### Backend Setup
```bash
cd backend
npm install
cp .env.example .env
# Edit .env with your database credentials
npm run dev
```

### Frontend Setup
```bash
cd frontend  
npm install
npm run dev
```

See INSTALLATION.md for detailed instructions.

## Features
- ✅ JWT Authentication
- ✅ Role-based access (Student/Faculty/Admin)
- ✅ Course Management
- ✅ Video Lessons with YouTube
- ✅ Exam System with Auto-grading
- ✅ Payment Integration (Razorpay)
- ✅ Email Notifications
- ✅ Progress Tracking
- ✅ Analytics Dashboard

## Tech Stack
- **Frontend**: React 18, Vite, Tailwind CSS
- **Backend**: Node.js, Express, Sequelize
- **Database**: MySQL
- **Auth**: JWT
- **Payments**: Razorpay
- **Email**: Resend
